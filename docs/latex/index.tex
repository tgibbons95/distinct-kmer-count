\hypertarget{index_autotoc_md0}{}\doxysubsection{About The Project}\label{index_autotoc_md0}
This project is meant to find the number of unique K-\/mers in the suffix tree using the BWT and Reverse BWT of a string. The output computed in this repo was found from a provided DNA segquence for the assignment.

Known implementation issues
\begin{DoxyItemize}
\item Issues finding the children of an internal node and decrementing the kmer count in constant time. The algorithm should still compute the value correctly, but is linear in finding the children. In the worst case, this algorithm will run O(n$^\wedge$2). With the provided input for this assignment, the sequence still ran quickly because it wasn\textquotesingle{}t a worst case scenario.
\end{DoxyItemize}
